About this project

Google App Engine and Spring MVC Quick Start is a simple way to quickly start developing your project with deployment to Google App Engine. It uses Objectify to read and write data using Google Datastore. There are 2 controllers written to handle requests to main page and news page.

What's already done here

Here you can find:

  • two controllers: one for main page and another for everything concerning news;
  • entity example for News entry
  • simple data access layer which uses Objectify. Methods that already have been implemented: save, delete, listObjectByPage, countAll, getUniqueEntity;
  • custom exception and code in controller for handling it;
  • *.jsp pages with JSTL and template usage;
  • restricted access for administrative part: adding news and deleting news;
  • working demo can be found here.
